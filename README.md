# Customer Segmentation Using K-Means Clustering

## Project Overview

This project focuses on customer segmentation using Machine Learning techniques to identify distinct groups of customers based on their purchasing behavior and demographic characteristics.

Customer segmentation helps businesses understand customer behavior, improve marketing strategies, increase customer retention, and optimize business decisions.

---

## Objective

The objective of this project is to:

- Analyze customer data.
- Identify meaningful customer segments.
- Apply K-Means Clustering to group customers.
- Visualize customer clusters.
- Generate actionable business insights and recommendations.

---

## Project Workflow

### 1. Data Collection

Customer dataset containing demographic and spending information was collected.

### 2. Data Cleaning

- Missing value handling
- Data validation
- Feature selection

### 3. Exploratory Data Analysis (EDA)

- Customer distribution analysis
- Income analysis
- Spending behavior analysis
- Correlation analysis

### 4. Feature Engineering

Relevant customer attributes were selected for clustering.

### 5. K-Means Clustering

The K-Means algorithm was applied to segment customers into distinct clusters.

### 6. Cluster Visualization

Customer groups were visualized using scatter plots and cluster diagrams.

### 7. Dashboard Development

Interactive Power BI dashboard was created for customer analysis.

---

## Machine Learning Model

### K-Means Clustering

K-Means is an unsupervised machine learning algorithm used to divide customers into groups based on similarity.

The model helps identify:

- High-value customers
- Potential customers
- Regular customers
- Low-engagement customers

## Key Insights

- Premium customers contribute significantly to business revenue.
- Certain customer groups require targeted marketing campaigns.
- Medium-value customers can be converted into premium customers.
- Low-spending customers need engagement and retention strategies.
- Customer segmentation improves marketing effectiveness.

## Business Recommendations

- Create personalized marketing campaigns.
- Focus on high-value customer retention.
- Develop loyalty programs.
- Offer targeted promotions for specific customer segments.
